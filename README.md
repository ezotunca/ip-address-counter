IP Address Counter

Bu küçük proje, log dosyalarıyla biraz pratik yapmak için hazırlanmıştır.

Program bir log dosyasını okur ve hangi IP adresinin kaç kere geçtiğini sayar.
Amaç, log mantığını anlamak ve tekrar eden IP’leri fark etmeyi öğrenmektir.

Dosyalar:
- sample.log : Deneme amaçlı log dosyası
- counter.py : IP adreslerini sayan Python dosyası

Bu proje tamamen öğrenme amaçlıdır.
